###### A program for [[Unix-like]] system, allow user to run a program or perform action with security privileges of another user, by default the [[superuser]].

> [!What is "sudo"?]
> - It originally stood for **"superuser do"**.
> - However, according to Linux manual pages, they define "su" as "substitute user", so we can understand it as **"substitute user, do"**.
