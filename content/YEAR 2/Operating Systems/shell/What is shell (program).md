- Shell, or operating system shell, is a **command-line interface**. The root history and how shell work is come from Unix.
- Basically shell from [[Unix]] can be use on [[Linux]] or Windows, same as Linux because it is [[Unix-like]]. But Windows shell cannot be use on Unix.

## Unix
- Thompson shell (1971)
- [[sh (Bourne shell)]] (1977)
- csh (C-like shell) (1978)
## Linux
- [[bash (Bourne-Again Shell)]] (1989)
## Windows
- [[cmd.exe]] (1993)
- [[PowerShell]] (2016)
- [[Windows Terminal]] (2019), which just a terminal emulator with multi-tab:
	- [[cmd.exe]]
	- [[PowerShell]]
	- [[sh (Bourne shell)]]
	- git bash
	- and more ...
